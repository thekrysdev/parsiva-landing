# Parsiva Landing Page

Marketing-Website für Parsiva — KI-gestützte Dokumentenverarbeitung für den deutschen Mittelstand.

## Stack

- Next.js 16 (App Router, Turbopack)
- Tailwind CSS v4
- TypeScript strict
- Deployed on Vercel (Frankfurt)

## Development

```bash
npm install
npm run dev
```

Läuft auf http://localhost:3000.

## Build

```bash
npm run build
npm run start
```

## Deployment

Auto-Deploy zu Vercel bei Push auf `main`.
